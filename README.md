# Notesapp-server

- Asynchronous server application to support Kotlin multi platform HTTP client apps
- REST APIs are written in Kotlin using KTOR Server framework
- MySQL is used as a database
- ktorm is used as ORM framework

Features
- Add Notes
- Get Notes
- Delete Notes
- Update Notes
